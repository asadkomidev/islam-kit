# Getting Started

## Installation

Install the packages you need:

```bash
# Prayer Times
npm install @islam-kit/prayer-times

# Qibla Direction
npm install @islam-kit/qibla

# Quran
npm install @islam-kit/quran
```

Or install all at once:

```bash
npm install @islam-kit/prayer-times @islam-kit/qibla @islam-kit/quran
```

## Package Managers

### npm
```bash
npm install @islam-kit/prayer-times
```

### yarn
```bash
yarn add @islam-kit/prayer-times
```

### pnpm
```bash
pnpm add @islam-kit/prayer-times
```

### bun
```bash
bun add @islam-kit/prayer-times
```

## Platform Support

All Islam Kit packages work across the JavaScript ecosystem:

| Platform | Support |
|----------|---------|
| Node.js | ✅ ESM & CommonJS |
| Browser | ✅ Via bundlers or CDN |
| React | ✅ Direct import |
| React Native | ✅ No native modules |
| Expo | ✅ Works out of the box |
| Next.js | ✅ Server & Client |
| Cloudflare Workers | ✅ Edge compatible |
| Deno | ✅ Via npm specifier |

## TypeScript

All packages include TypeScript definitions. No additional `@types` packages needed.

```typescript
import type { PrayerTimes, Coordinates } from '@islam-kit/prayer-times';
import type { QiblaResult } from '@islam-kit/qibla';
import type { Surah, Reciter } from '@islam-kit/quran';
```

## ESM & CommonJS

All packages support both ESM and CommonJS:

### ESM
```javascript
import { getPrayerTimes } from '@islam-kit/prayer-times';
```

### CommonJS
```javascript
const { getPrayerTimes } = require('@islam-kit/prayer-times');
```

## Bundle Sizes

Islam Kit is designed for minimal bundle impact:

| Package | Size (gzipped) |
|---------|----------------|
| @islam-kit/qibla | ~1 KB |
| @islam-kit/prayer-times | ~5 KB |
| @islam-kit/quran (utilities) | ~2 KB |
| @islam-kit/quran/data | ~800 KB |

The Quran data is optional - import from `@islam-kit/quran/data` only if you want bundled JSON data.
